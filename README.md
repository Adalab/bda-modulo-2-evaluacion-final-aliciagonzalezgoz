# Evaluación Final Módulo 2: SQL 🎬

Repositorio de la evaluación final del Módulo 2 del Bootcamp de Análisis de Datos de Adalab, realizada por **Alicia González**.

## Descripción

Colección de ejercicios de SQL sobre la base de datos **Sakila**, una base de datos de ejemplo que simula una tienda de alquiler de películas. Contiene consultas de distinta complejidad que cubren desde queries básicas hasta subconsultas y múltiples JOINs.

## Tecnologías utilizadas

- **MySQL** — Motor de base de datos
- **Base de datos Sakila** — Base de datos de ejemplo oficial de MySQL

## Requisitos previos

- MySQL instalado en tu máquina. Puedes descargarlo desde [mysql.com](https://dev.mysql.com/downloads/).
- La base de datos Sakila instalada. Puedes descargarla desde [dev.mysql.com/doc/sakila](https://dev.mysql.com/doc/sakila/en/).

## Cómo arrancar el proyecto

1. Clona el repositorio:
   ```bash
   git clone https://github.com/Adalab/bda-modulo-2-evaluacion-final-aliciagonzalezgoz.git
   ```

2. Abre MySQL y asegúrate de tener la base de datos Sakila cargada:
   ```sql
   USE sakila;
   ```

3. Abre el archivo `Examen_Modulo_2_Alicia_Gonzalez.sql` en tu cliente MySQL (MySQL Workbench, DBeaver, etc.) y ejecuta las queries.
